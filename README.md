# AI Agent Study

6주 동안 AI Agent를 학습하고 직접 설계·구현해보는 프로젝트 스터디 기록입니다.

## 목표

- 업무/삶에서 자동화할 여지가 있는 문제를 직접 정의한다
- Agent를 활용한 해결책을 설계하고 구현한다
- Agent의 성과를 측정하고 개선한다

## 진행 현황

| 주차 | 주제 | 상태 |
|------|------|------|
| 1주차 | AI Agent 개념 학습 | ✅ 완료 |
| 2주차 | 하네스/에이전트 툴 분석 | ✅ 진행중 |
| 3주차 | 문제 정의 / 설계 | 🔜 예정 |
| 4주차 | 구현 | 🔜 예정 |
| 5주차 | 성능 측정 / 개선 | 🔜 예정 |
| 6주차 | 최종 정리 / 공유 | 🔜 예정 |

## 폴더 구조

```
week1/   - AI Agent 개념 학습
week2/   - 도구 분석 및 문제 후보 정리
week3/   - 문제 정의 및 Agent 설계
week4/   - 구현
week5/   - 성능 측정 및 개선
week6/   - 최종 정리
```
